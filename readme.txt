Adds Creamailer mailinglist service integration for WooCommerce customers

== Description ==
Adds Creamailer mailinglist service integration for WooCommerce customers. You need an creamailer account to use this plugin

== Install ==
1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate plugin using Wordpress admin-panel.
3. Adjust your Creamailer account settings under Settings->Creamailer.
